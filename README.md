# OPT-IN プロジェクト

マーケティング・セミナー設計に関する資料をまとめたリポジトリです。

---

## フォルダ構成

```
opt-in/
├── 01_セミナー分析/      # デザインBCフロントセミナーの分析・データ
├── 02_参考資料/          # セミナー設計ガイド・AIブートキャンプ・VSL資料
├── 03_セミナー構成/      # バックエンドセミナーの構成・スライド
├── 04_メンズライズ/      # モニター募集分析・YouTube台本
├── 05_ビジ系ストーリー/  # ストーリー戦略
└── README.md
```

---

## 各フォルダの内容

### 01_セミナー分析/
デザインBCフロントセミナーに関する分析資料・データ。

| ファイル | 内容 |
|----------|------|
| `デザインBCフロントセミナー分析.md` | セミナーの問題分析・改修指示・優先度まとめ |
| `デザインBCフロントセミナー文字起こし.md` | セミナー資料（スライド）の文字起こし |
| `ブートキャンプ数値管理シート - セミナー所感.csv` | スタッフからのセミナー所感 |
| `１分アンケート（回答） - 集計.csv` | 参加者・不参加者へのアンケート集計 |

### 02_参考資料/
セミナー設計の参考資料・ナレッジ。

| ファイル | 内容 |
|----------|------|
| `売れるセミナー構成ガイド.md` | セミナー設計の9段階フレームワーク + 10原則 |
| `セミナー作成FB集.md` | セミナー作成のフィードバック集 |
| `セミナー文字起こし.md` | 売れているセミナー2本の文字起こし（参考用） |
| `たくむAIセミナー.txt` | たくむAIセミナーの文字起こし |
| `AIブートキャンプ概要.md` | AIブートキャンプの概要 |
| `VSL/告知用VSL/` | 告知動画の台本・文字起こし |

### 03_セミナー構成/
バックエンドセミナーの構成・スライド文言。

| ファイル | 内容 |
|----------|------|
| `バックエンドセミナー構成_最終版.md` | バックエンドセミナーの最終構成 |
| `バックエンドセミナースライド文言.md` | スライドの文言詳細 |
| `スライドモック.html` | スライドのモックアップ |

### 04_メンズライズ/
メンズライズ関連のマーケティング資料。

| ファイル | 内容 |
|----------|------|
| `モニター募集分析レポート.html` | モニター募集の属性分析レポート |
| `リョータくん/リョータch分析ドキュメント.txt` | チャンネル分析 |
| `リョータくん/YouTube台本/` | YouTube動画の台本（4本） |

### 05_ビジ系ストーリー/

| ファイル | 内容 |
|----------|------|
| `ストーリー戦略.md` | ビジネス系ストーリー戦略 |
