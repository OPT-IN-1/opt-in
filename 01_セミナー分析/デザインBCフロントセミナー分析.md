## 1. BCの概要

### 基本情報

| 項目 | 内容 |
|------|------|
| **サービス名** | SNSデザインブートキャンプ |
| **コンセプト** | 短期間でのスキル獲得を目的にした講座 |
| **期間** | 3日間 |
| **価格** | 19,800円（定価79,800円の75%OFF） |
| **販売形式** | 買い切り型（ローンチ期間のみ販売） |
| **決済手段** | MOSH |
| **視聴期間** | 半永久（購入後いつでも視聴可能） |
| **開催日程** | 2026年2/13(金)〜2/15(日) |

### ターゲット

| カテゴリ | 詳細 |
|----------|------|
| **副業を始めたい人** | 会社員で副収入を得たい層 |
| **何を始めればいいかわからない人** | 副業したいが方向性が定まっていない層 |
| **SNSで稼げると思っていない人** | SNSマネタイズに懐疑的な層 |
| **働き方を変えたい人** | 時間と場所にとらわれない働き方をしたい層 |
| **発信者・インフルエンサー層** | 自分の投稿デザインに自信がない層 |

### サービス内容（3本柱）

#### ① 動画カリキュラム
- 1日1本、自動公開（0時に公開）
- 基礎的なスキル獲得＋課題
- MOSHで24時間視聴可能、半永久で視聴可能

#### ② メンターのマンツーマンサポート
- 1人1人に個別メンターが担当
- 毎日の課題に対する返信＋日報への**動画フィードバック**
- 提出期限：当日23:59までに提出 → 翌日12:00までに添削
- **※この動画添削が他社にない強み・満足度が高いポイント**

#### ③ Zoom作業会
- 平日1回、土日は1日2回開催（全5回）
- 実際に運用代行の案件を取っている現役講師が登壇
- カリキュラムで出た疑問をリアルタイムで解消

### カリキュラム（デザインの場合）

| 日程 | 内容 |
|------|------|
| **DAY1** | 既存デザインのトレース作業① |
| **DAY2** | 既存デザインのトレース作業② |
| **DAY3** | オリジナルデザイン作成（案件獲得の擬似体験） |

### 他社との差別化

| 比較項目 | 他社 | BC |
|----------|------|----------------|
| **価格** | 10万円以上 | **19,800円** |
| **動画講義** | ○ | ○ |
| **課題** | 自分でこなすだけ | ○＋**個別フィードバック** |
| **個別サポート** | テキストのみ or なし | **動画での個別添削** |
| **リアルタイム講義** | 高単価商材のみ | ○（Zoom作業会） |

### 運営の裏側の目的

| 目的 | 詳細 |

## 2. セミナーの位置づけと導線

```
① SNS（Instagram/YouTube）で認知
    ↓
② LINE登録
    ↓
③ 1day勉強会セミナー参加 ← ★このセミナー
    ↓
④ BCの内容を知る
    ↓
⑤ 自分で購入意思を決定（※セールスマンなし）
    ↓
⑥ MOSHで決済・購入完了
```

### 注意事項
- **BCでは収益化は学べない**（3日間で収益化は現実的でない）
- ゴールは**ゼロからスキル獲得**
- 収益化方法は**ロードマップセミナー**（同封）で解説

## 5. データから見える問題点

### 5-1. スタッフ所感からの問題

> 参加者の多くは「副業選び」以前に、「デザインの質をどう上げるか」「営業やポートフォリオをどう作るか」といった**実務フェーズに近い悩み**を持っている。

→ **セミナーの前半が「参加者がすでに知っていること」に時間を使いすぎている**

> 「この3daysのどの部分で」「どんな変化が起きるのか」をより明確に言語化できると、自動応答に進む納得感がさらに高まる

→ **BCの中身が抽象的で、参加者が「自分がどう変わるか」をイメージできていない**

### 5-2. アンケートから見える未購入理由

#### BC未購入理由TOP5

| 順位 | 理由 | 割合 |
|------|------|------|
| 1位 | **忙しく、取り組める自信がなかった** | 25.58% |
| 2位 | 料金に対する価値をまだイメージできなかった | 13.95% |
| 3位 | まずは情報収集から始めたい | 11.63% |
| 4位 | やりきれるか不安だった | 6.98% |
| 5位 | 他の講座と比較して迷っている | 4.65% |

**→ 1位+4位=32.56%が「やりきれるか不安」。これが最大の購入障壁。**

#### アンケート自由回答で繰り返し出たフィードバック

- 「購入後の内容がはっきりわからなかった」
- 「3日間でどんな事を学べるかもう少し詳しく説明してほしい」
- 「スクールの必要性を押すばかりで肝心の内容があまり分からなかった」
- 「実際に仕事を開始している人の生の声を聞きたかった」
- 「銀行振込ができない」（複数）
- 「進行が早い」「ペースが速い」

### 5-3. 録画分析から見える問題

| 問題 | 詳細 |
|------|------|
| **時間配分の偏り** | 副業比較+SNSデザイナー説明+99%失敗 = 約50分。BC紹介 = 約15分 |
| **恐怖訴求過多** | 借金事例、99%失敗、機会損失を繰り返し。「やりきれるか不安」を増幅するリスク |
| **BCの具体性不足** | 1日スケジュール・Before/After・価値一覧は追加済みだが、FB動画の実物はまだ見せていない。また、追加したスライドもBC紹介パートの時間自体が短いため十分に伝わりきっていない可能性 |
| **クロージングの精神論** | 迷っている人への対応が「勢いが大事」「後回しにしないで」にとどまる |

## 6. 改善の方針と具体策

### 根本方針：3つの軸

| 軸 | 内容 |
|----|------|
| **① 削る** | 参加者がすでに知っていることに時間を使わない |
| **② 見せる** | BCの価値を言葉ではなく実物で体験させる |
| **③ 最大の購入障壁に直接答える** | 「やりきれるか不安」に事実で安心させる |

### 前回の改善で実施したこと

前回の改善案では18枚のスライド追加が提案されたが、**実際に実装されたのは以下の3点のみ**：

| 実装済み | 内容 |
|----------|------|
| ✅ | 1日のリアルなスケジュール（BCの1日の時間配分を提示） |
| ✅ | 受講生のBefore/After（DAY1→DAY3のデザイン変化を提示） |
| ✅ | 19,800円で手に入るもの（価格提示時の具体的な価値一覧） |

#### 未実装のまま残っている改善案

| 未実装 | 内容 |
|--------|------|
| ❌ | 「一人でやる」vs「BC」の対比スライド |
| ❌ | 3日目の「擬似体験」の強調スライド |
| ❌ | FBの価値を言語化するスライド |
| ❌ | 反論処理パートの強化（6枚） |
| ❌ | クロージングで「未来」を強調するスライド |
| ❌ | インフルエンサー向け訴求スライド |

### 実装済み3点でも数値が上がらなかった理由

実装した3点はいずれも「BC紹介パート内のスライド追加」であり、方向性は正しい。
しかし数値が動かないのは、**問題がBC紹介パートの中身だけでなく、セミナー全体の構造にある**ため。

具体的には：
1. **BC紹介に到達する前の50分間が長すぎ**、参加者の集中力・温度が下がっている
2. **恐怖訴求（99%失敗、借金事例）が不安を増幅**し、追加スライドの安心効果を打ち消している
3. BC紹介パート自体が**全体の約15分しかなく**、追加スライドを入れても十分に伝わる時間がない
4. クロージングで迷っている人への対応が**依然として精神論**（「勢いが大事」）にとどまっている

**→ スライドの追加だけでは解決せず、前半の大幅圧縮によるBC説明時間の確保が必要。**

## 7. スライド単位の改修指示

### 改修①：副業比較パート（P78〜P95）→ 大幅圧縮

**現状**：6職種×各3枚 = 18枚（約15分）

**変更**：1枚のまとめスライドに圧縮

```
【差し替えスライド】（P78の位置に1枚）
よくある副業、実は落とし穴だらけ

ブログ → 結果が出るまで1年以上。100記事書いても収益ゼロはザラ
プログラミング → 学習に半年〜1年。AIの台頭で将来性に不安
せどり・物販 → 初期投資が必要。在庫リスクあり
Webライティング → 競合が多く文字単価1円。時給500円も
動画編集 → 高スペックPC必須。単価下落中
Webデザイン → 感性が必要。単発が多く安定しない
```

→ P79〜P95を削除。P96（正しい副業選びの条件）につなげる。
→ **約15枚削減**

| P114（カフェ系の型） | P113に統合可能 |

| P126〜P128（0.5秒で判断の話） | 1枚にまとめる |
| P129〜P134（市場規模〜企業活用率） | 2枚にまとめる |
