# リールアカBAN対策（台本）

## 抽象的な考え方（前提）

### 1. 視聴者目線で通報されないかを判断する
- **これが一番大事**
- どのワードがダメとかより感覚的なところ
- 同じテーマでもBANされるもの・されないものがある
- → 通報されるかされないかが一番でかい

### 2. 危険ではないワードで同じくらい当たり感あるならそれを使う
- クリーンで当たり感あるワードを追求する姿勢が評価される
- 「これ大丈夫ですか？」「どういう感覚になりますか？」を追求する

### 3. リスク判断
**危険度 × 当たった場合の恩恵** を天秤にかける

- 売り上げ規模（大きいほど慎重に）
- BANされてもまた伸ばせるかどうか
- 属人性・インパクトがあれば復帰しやすい

## 危険なワード一覧

### お金系ワード
- お金、副業、稼げる、年収、月収
- 収益、収入、売り上げ、利益
- 毎月、時給、日給
- 在宅ワーク

### 簡易性ワード
- 勝手に、自動で、隙間時間で
- 誰でも、全員が、簡単に

### 断定ワード
- 確実に、絶対に

## 危険度の考え方

**単体では大丈夫でも、組み合わせると危険**

例：
- 「副業」だけ → OK
- 「副業」+「100万」+「誰でも」+「自動で」 → 危険

→ 危険ワードが多いほど、通報されやすいニュアンスに近づく

### 最悪の組み合わせ例
```
ミスるアプリだけで100万円
確実に自動化
隙間時間で
```
→ めちゃくちゃ危険

## 対策

### 1. 台本内に入れる危険ワードは最小化 + 当たり感も担保
- 1個も使わないのはきつい
- フックのところだけ強めに入れて、あとはマイルドにする
- バランスが大事

### 2. ボキャブラリーを増やす
- 同じ意味合いの文章に対して、ワードのパターンを大量に持つ
- ChatGPTで言い換え100個出す → 1番クリーンで当たり感あるものを選ぶ
- 「その意味合いだったらそのワードしかない」レベルを目指す

### 3. 文字化けさせてAI検知を避ける（非推奨）
- 例：副業 → ﾌｸ業
- **本質的ではない**
- 文字化けさせても、全体の雰囲気が怪しければ通報される
- 逆に怪しさが増すこともある

---

## 美しい台本とは

**クリーンで当たり感ある** = 最高の台本

- 競合の上位互換になっている
- かつ危険ワードは最小化されている
- 弱くなっていない

### ダメな例
- 危険ワードを1個も使わず弱くなっている
- 競合の上位互換になっていない

### 良い例
- 競合の上位互換になっている
- 危険ワードを最小化している
- 全体の雰囲気がクリーン

## リスクバランスの判断基準

| 状況 | 判断 |
|-----|------|
| 新規立ち上げアカ | 多少攻めてもOK（失うものがない） |
| 月1000万以上売れてるアカ | かなり慎重に |
| 属人性・インパクトある人 | BANされても復帰しやすい |
| 属人性ない・後発アカ | BANされるときつい |

---

## パクリについて

### 基本ルール（ケースバイケース）

1. **視聴者目線でパクリとなるかどうかで判断**

2. **15文字以上連続して同じ文章を使用しない**

3. **構成や話すノウハウの順番が全く一緒はNG**
   - 1個〜2個違うやつ入れて順番入れ替える

4. **全体を通して8割ぐらい文章が変わっているようにする**
   - 内容は同じだけど言い換えてあるとかも含めて8割変わってたらOK

## まとめ

1. **視聴者目線で通報されないか**が一番大事
2. 危険ワードは最小化、でも当たり感は担保
3. クリーンで当たり感あるワードを追求する姿勢
4. ボキャブラリーを増やして最適解を出す
5. リスクと恩恵を天秤にかけて判断
6. パクリは視聴者目線で判断、8割変える
