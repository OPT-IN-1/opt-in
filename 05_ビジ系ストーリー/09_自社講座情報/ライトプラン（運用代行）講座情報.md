# **インスタ集客用 フロントコンテンツ ナレッジ**

## **1\. 講座情報**

### **1-1. 講座の目的・概要**

* **運用代行を学ぶ講座**（セミナー中では「リバティ」と呼ばれる）

* "未経験から最短1〜3ヶ月で収益化（月10〜20万円）"を目標とする。

* **特徴**:

  * デザイン・動画編集・営業・運用の基礎〜応用を学べる

  * 最短1ヶ月で案件獲得する人もいる

### **1-2. カリキュラム例**

* **6ヶ月プログラム**（セミナー内で紹介）

  1. **1ヶ月目**: キャンバなどでフィード投稿のデザイン基礎

  2. **2ヶ月目**: リール動画編集（スマホアプリで作る）

  3. **3ヶ月目**: 営業スキル（契約のとり方、価格設定など）

  4. **4〜6ヶ月目**: 実案件対応、単価UPの応用スキル（TikTok運用、LINE公式など）

### **1-3. サポート体制**

* 5人の専属講師が1人の受講生をサポート（営業、デザイン、SNSマーケ等）

* 個別メンターがつき、毎日フィードバック（日報提出など）

* 24時間質問OK、週1回ズーム勉強会等

* 6ヶ月で月20万円到達しなければ**サポート延長**

### **1-4. 受講費・支払**

* 詳細金額は個別相談で案内

* 一括/分割あり、ローン可

* 「最終的に成果（契約獲得など）を重視する保証制度を用意」

## **2\. フォロワー・顧客情報**

### **2-1. どんな企業や顧客がターゲットか**

* セミナー中では「**中小企業**がインスタ担当を探している」「美容院や歯医者、サロン、飲食店、個人事業主なども多い」

* **月10〜20万円**の外注費でSNS運用を任せたいが、社内にできる人材がいない企業が多数

### **2-2. フォロワー（スクール生）の属性**

* **受講生の約4割が主婦、他に会社員・フリーランス、老若男女**

* 共通点：

  * 時間が限られている（子育て、本業等）

  * 未経験からのスタートが大半

  * 短期間で安定収入を得たい

### **2-3. お客さま（セミナー参加者）の悩み**

* 副業を考えつつも「何をやっていいか分からない」「時間やスキルに不安」

* 会社に依存せず「在宅ワークで数万円〜数十万円稼ぎたい」

* 過去に動画編集やアフィリに手を出したがうまくいかなかった

## **3\. 受講生事例・成功体験**

### **3-1. ビフォーアフター例**

* **元ニート→月7桁収益**

* **会社員（介護、看護、建設業、etc.）→3ヶ月で月20〜45万円**

* 子育て中でも1〜2時間/日の作業で契約獲得

### **3-2. 具体的ストーリーズ**

* **ママの例**

  * 2ヶ月目で合計35万円分の契約

  * 子どもが昼寝している間に投稿作成、夜にZoomで確認→短期で成果

* **会社員の例**

  * 本業が終わってから1〜2時間取り組み→1ヶ月で月9万円の長期契約

* **主婦でPC苦手→月10万円安定**

### **3-3. 事例の訴求ポイント**

* 少ないフォロワーでもOK

* 運用未経験でもOK

* 3ヶ月で安定収益（月20万）を得られる→「人生変わった」

## **4\. よくある不安・質問のQ\&A**

1. **パソコンが必要？スマホだけでもいい？**

   * 基本はスマホでも作業可だが、PCがあるとリサーチや資料作成が効率的。

2. **営業って飛び込み？ 怖くない？**

   * クラウドワークスやWantedlyなどWEB完結の案件募集あり。スーツ着て企業に突撃は不要。

3. **本業や育児があるが時間がとれない**

   * 1日2時間は確保。早朝・深夜の隙間でもOK。

4. **フォロワーゼロ＆デザイン未経験、ほんとに大丈夫？**

   * テンプレートやキャンバで短期間で作れる。フォロワー数は関係なく契約可。

5. **短期間で稼げるなんて怪しくない？**

   * 企業がSNS人材を探す需要が大きい。月10〜20万の相場で契約しやすい仕組みがある。

## **5\. 痛み・失敗ポイント**

* **なぜうまくいかない人が多い？**

  * 無料情報だけで自己流→やり方がブレる

  * スキルやロードマップがないまま高額案件を狙う→挫折

  * 「何をやるべきか」不明で悩み続けて進めない

* **投資・物販・アフィリなどの副業**

  * 時間かかる、初期費用や在庫リスク、報酬が小さい…で挫折しやすい

* **途中でモチベが切れてやめる**

  * サポートや仲間・コーチがいないと続かない

| 副業 | デメリット・難点 |
| ----- | ----- |
| **投資** | 再現性低。失敗リスク高。初心者には難易度大 |
| **アフィリ** | 単価が小さい（1件500円など）。月10〜20万に達しにくい |
| **動画編集** | ライバル多く単価ダウン、PCスペック・ソフト費用も必要 |
| **コンテンツ販売** | スキル・実績が必須。初心者はハードル高い。 |
| **運用代行** | 月10〜20万円、安定契約、在庫リスクなし、ライバル少 |

# **まとめ**

このセミナーでは「運用代行で月10〜20万円を目指す」ストーリーが主軸でした。  
 **フロント発信やLP**では、下記のポイントを意識してコンテンツを構成すると、見込み客の関心や不安をフォローしやすくなります。

1. **講座情報**：学習ステップ、サポート体制、保証制度をわかりやすく。

2. **フォロワー／顧客像**：需要のある中小企業が多い、月10〜20万の継続報酬が狙える。

3. **受講生のビフォーアフター**：具体的数字・期間つきで事例紹介（主婦、会社員、未経験など）。

4. **よくある質問**：時間・PCスキル・営業方法・不安を想定Q\&A形式で解消。

5. **失敗ポイント**：無料情報頼り、やり方がブレる… →正しい学び＋継続が大事。

6. **他副業との比較**：物販・投資・アフィリなどの弱点→運用代行の優位性を対比。

7. **仕事イメージ**：1日の流れ、作業内容、月◯万円の相場感を簡単に提示。

8. **想定の未来**：在宅で仕事、会社に縛られない、自分が成長する喜び、家族との時間。

9. **主催者自身のストーリー**：過去の挫折〜短期で稼げた背景→信頼と共感を得る。

10. **多様性**：性別・年齢・職業を問わず成功する例を示し、ターゲットが「自分にもできそう」と感じる。

これらの要素を中心にフロント発信を組み立てれば、**セミナー内容の"刺さる"部分**を短く・わかりやすく届けることができ、見込み客の疑問や不安を先回りでカバーしながら興味を引くコンテンツを作りやすくなるでしょう。
